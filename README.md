# Healing Foundation Rehab Centre Website

A premium rehabilitation and wellness centre website built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Editing Content

Most website content lives in `data/site.ts`. Update treatments, facilities, FAQs, testimonials, blog posts, contact details, and navigation there.

## Deployment

This project is ready for Vercel.

```bash
pnpm build
```
